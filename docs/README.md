# contexta Docs

Developer documentation for [contexta](https://github.com/contexta-dev/contexta) — the memory intelligence layer for AI agents.

Built with [Nextra](https://nextra.site) + MDX.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm run start
```

## Structure

- `src/app/quickstart/` — 5-min quickstart
- `src/app/concepts/` — core concepts (observations, memories, retrieval)
- `src/app/guide/` — integration guides (OpenAI, LlamaIndex, Anthropic, LangChain, custom)
- `src/app/reference/` — API, SDK, CLI reference
- `src/app/examples/` — full agent examples
- `src/app/changelog/` — release notes
- `src/app/pricing/` — pricing plans
