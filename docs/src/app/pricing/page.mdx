## Overage Rates

| Dimension | Rate |
|-----------|------|
| Observations | $0.002 each |
| Retrievals | $0.0001 each |
| Reranks | $0.003 each |
| Storage (per 1k memories/mo) | $0.50 |

Overage is billed at end of cycle. Hard caps can be enabled per API key to prevent surprises.

## Annual Discount

All plans are available with a 2-month discount when billed annually:

| Plan | Monthly | Annual |
|------|---------|--------|
| Builder | $29 | $290/yr |
| Scale | $99 | $990/yr |
| Dedicated | $299+ | $2,990+/yr |

## Enterprise

Need dedicated infrastructure, VPC peering, SOC 2 reports, or custom SLAs? [Contact us](mailto:enterprise@contexta.dev) for a custom plan ($5k–$25k+/month).

## What's Free

The following are never metered:

- API key management (create, rotate, revoke)
- Dashboard access
- Documentation, SDK downloads
- Audit log (within retention window)
- Memory deletion
- Data export (one quarterly full export per tier)
- Policy and schema management
