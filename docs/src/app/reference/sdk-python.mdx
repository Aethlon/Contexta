# Python SDK Reference

Full API surface for `contexta-client`.

## Installation

```bash
pip install contexta-client
```

## Classes

### `contexta`

Sync client.

```python
class contexta:
    @classmethod
    def from_env(cls) -> contexta: ...

    def __init__(
        self,
        *,
        api_key: str,
        base_url: str = "https://api.contexta.dev",
        timeout: float = 30.0,
        max_retries: int = 3,
        telemetry: bool = True,
        enable_buffer: bool = True,
    ): ...
```

### `Asynccontexta`

Async client. Same constructor and method signatures, but all methods are async.

```python
class Asynccontexta:
    @classmethod
    def from_env(cls) -> Asynccontexta: ...
```

### context

```python
def context(
```

## Data Types

```python
@dataclass
class Context:
    user_profile: UserProfile | None
    active_projects: list[Project]
    preferences: list[Preference]
    goals: list[Goal]
    recent_events: list[Event]
    relevant_memories: list[ScoredMemory]
    token_usage: TokenUsage
    cache_hit: bool
    request_id: str

    def to_system_prompt(self, *, template: str | None = None) -> str: ...
    def to_messages(self) -> list[dict]: ...
    def to_markdown(self) -> str: ...
    def to_dict(self) -> dict: ...

@dataclass
class ScoredMemory:
    memory_id: str
    memory_type: str
    title: str
    content: str
    score: float
    score_breakdown: dict
    tags: list[str]
    created_at: str

@dataclass
class Explanation:
    memory: Memory
    extraction_source: list[dict]
    scoring_breakdown: dict
    supersession_history: list[dict]

@dataclass
class Memory:
    id: str
    memory_type: str
    title: str
    content: str
    confidence: float
    importance: float
    tags: list[str]
    memory_state: str
    is_pinned: bool
    is_archived: bool
    created_at: str
    last_accessed_at: str | None
```
